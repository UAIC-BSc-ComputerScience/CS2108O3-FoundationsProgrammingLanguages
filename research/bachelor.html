Bachelor projects - 2017

  1. Basescu Andrei Costel
    • Project: Web course/lecture manager
    • Features (ro):
      • aplicatie mobile /web
      • profesorii si studentii vor avea cont
      • profesorii pot face urmatoarele :
        • gestiunea cursurilor (upload,delete,absolute adica pentru fiecare an sa se faca o arhiva )
        • pot pune note private fiecarui student
        • pot face anunturi referitoare la materie,pot raspunde la intrebarile studentilor.
      • studentii pot:
        • vizualiza in format pdf,ppt,word
        • trimite mesaje private catre profesori
        • vizualiza mesaje publice(wall) per disciplina
        • vizualiza doar notele lor
      • atunci cand posteaza un profesor fiecare student inscris la materia predata de profesorul respective va fi anuntat pe mail
      • profesorii incarca materialele pentru materie si aplicatia va posta automat materialele necesare pentru fiecare saptamana
  2. Avram Bogdan
    • Project: Online tool platform
    • Short description (ro):
      • rulare online a unor tool-uri/compilatoare (e.g., gcc, java, k)
      • administrare: adaugare/stergere tool-uri, editare si management fisiere (create, save, search, autocomplete, syntax highlighting), linie de comanda (comenzi filtrate - fara a afecta sistemul)
  3. Gabor Ramon

  4. Pascaru Stefan
    • Project: Collaborative learning
    • Short description: an online platform where users can work together in order to acquire IT knowledge
      • Users can start topics about a project, find other users interested in the subject and work for it together. Users can share resources: code, tutorials, documentation; comment or appreciate the quality of resources.
      • Moreover, users can share tutorials or other resources without being part of a project.
      • For every user, the application will create a resume, so the companies can find prospective future employees.
  5. Gabor Iuliana (progress: 10%)
    • Project: Semantics based test generation
    • Short description: automatic (language independent) test generation based on the semantics of programming languages encoded in K Framework.
  6. Saramet Stefan
    • Project: Online evaluation platform
    • Short description:
      • a platform for students' evaluation and for teachers' seminars/laboratories management.
      • main features:
        • for students: personalized view (e.g. grades,activity,trajectory), grades estimation, exam simulator (with question that can occur in the real exam), exam reviewer(shows the students' final exams and their faults).
        • for teachers: evaluation manager (tests prepared for each seminar/laboratory), autocorecter (calculates the tests' grades and marks them into a catalogue), presence tracker, personalizer for each course taught by a teacher, bonus calculator.
