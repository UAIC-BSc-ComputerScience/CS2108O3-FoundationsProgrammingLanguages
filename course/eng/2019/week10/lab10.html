Laborator 10

Exerciții propuse:

  1. (1 pct.) Adăugați la semantica small-step a expresiilor booleene din imp.v operațiile or și equal (egalitate peste AExp). Apoi arătați că or, equal și and se comportă corect pe următoarele exemple:
Notation "A ==' B" := (bequal A B) (at level 53).
Example bex_and :
  forall sigma,
    band (3 ==' 5) (5 <=' 3) ={ sigma }>* bfalse.
Admitted.

Example bex_or :
  forall sigma,
    bor (3 <=' 5) (5 <=' 3) ={ sigma }>* btrue.
Admitted.
  1. (1 pct.) Adăugați la semantica imp.v instrucțiunea par care are rolul de a executa două instrucțiuni în paralel:
Inductive Stmt :=
...
| par : Stmt -> Stmt -> Stmt.

Notation "S ||| S'" := (par S S')
                         (at level 62, left associativity).

Completați semantica acestei instrucțiuni în definiția lui eval:

Inductive eval : Stmt -> State -> Stmt -> State -> Prop :=
...
| epar1 : forall state state' s1 s2 s1',
    s1 -{ state }->[ s1' ; state' ] ->
    (s1 ||| s2) -{ state }-> ... 
| epar2 : ...
| epar : ...

Demonstrați că:

Definition par_loop :=
  (y ::= 1) ||| (while (y ==' 0) (x ::= x +' 1)).

Example x_remains_zero :
  forall sigma,
    sigma y = 0 ->
    sigma x = 0 ->
    exists sigma',
      par_loop -{ sigma }>*[ skip ; sigma'] /\
      sigma' x = 0.
Admitted.
  1. Bonus (1 pct.) Demonstrați că:
    Example x_can_be_1 :
forall sigma,
 sigma y = 0 ->
 sigma x = 0 ->
 exists sigma',
   par_loop -{ sigma }>*[ skip ; sigma'] /\
   sigma' x = 1.
Admitted.
