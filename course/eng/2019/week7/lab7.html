Laborator 7

Exerciții propuse:

  1. (1 pct.) Adăugați la sintaxa limbajului discutat la curs (imp.v) instrucțiunile if then și if then else. Adăugați semantica celor două instrucțiuni și testați semantica pe câte un exemplu. Pentru testare veți formula și demonstra lemele corespunzătoare în aceeași manieră în care s-a discutat la curs.

  2. (1 pct.) Demonstrați că limbajul rezultat este determinist.

Sugestii pentru rezolvare:

  • Citiți cu atenție formularea proprietății de mai jos și rulați pas cu pas demonstrația pentru a înțelege fiecare etapă:
Lemma aeval_is_deterministic:
  forall aexp st n n',
    aexp =[ st ]=> n ->
    aexp =[ st ]=> n' ->
    n = n'.
Proof.
  induction aexp; intros;
    inversion H; inversion H0;
      subst; auto.
  • assert (IH1: i1 = i0). eapply IHaexp1; eauto. assert (IH2 : i2 = i3). eapply IHaexp2; eauto. subst. reflexivity.
  • assert (IH1: i1 = i0). eapply IHaexp1; eauto. assert (IH2 : i2 = i3). eapply IHaexp2; eauto. subst. reflexivity. Qed.
  • Formulați și demonstrați o lemă asemănătoare pentru expresiile booleene.
  • Formulați și demonstrați o lemă asemănătoare pentru instrucțiuni.
